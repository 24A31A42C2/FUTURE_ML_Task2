# FUTURE_ML_Task2

## Store Sales Analysis using Python

### Project Overview

This project analyzes the Store Sales dataset using Python. It loads the dataset, checks dataset information, identifies missing values, calculates average daily sales, and visualizes daily sales using a line graph.

### Technologies Used

* Python
* Pandas
* Matplotlib

### Dataset

* train.csv

### Features

* Load and inspect the dataset
* Check missing values
* Calculate average daily sales
* Visualize daily sales trend using a line chart

### Output

* Dataset summary
* Missing values report
* Average daily sales
* Daily sales graph
